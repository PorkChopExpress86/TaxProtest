Harris County Property Lookup

Search for property information using account number, owner name, street name, or zip code.

Enter at least one search criteria to download property data. The system will generate an Excel (.xlsx) file with matching properties that you can open in Excel.

  • Account Number: Enter the first 5-6 digits to find properties in your area.
  • Owner Name: Enter part of the owner name (case-insensitive).
  • Street Name: Enter partial street names. Combine with zip code for better results.
  • Zip Code: Use alone to get all properties in an area, or combine with street name.
